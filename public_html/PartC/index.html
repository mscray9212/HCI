Background and Layout

A check between test team members was done to identify user friendliness and to ensure that the background was appealing. Also, the images were tested for relevancy and effectiveness.
RESULT: Background images and layout were acceptable.

Text

A test for readability was conducted on the website prototype. Test parameters included: literacy, spacing, fonts, understanding, and to ensure that any material listed was appropriate for the site.
RESULT: Textual context was acceptable.

Form

Overall form of the website and ease of use were tested. The form should be relative to the organization (permaculture). Since the user might not be computer savvy, the form should be navigatable for all users.
RESULT: Navigation bar was determined to be functionally unappealing.

Links

All test members checked to make sure that there were no 'broken' links and that all links directed to the correct destination. Also, all links were tested for topic associativity.
RESULT: Links were acceptable.

Browser Compatibility

SkwiuFarms was tested in four main internet browsers (Google Chrome, Mozilla Firefox, Internet Explorer, and Safari). Also, checks were done between different mobile browsers.
RESULT: Browser compatibility was acceptable.
