Concept Design: The design will detail an abstract of how the website should be created. This should include multiple media options to best

convey what an experience at the Japanese Culture Center will be like.


Website Plan: The plan of the website will display a CSS3 template outlying different media types used for the JCC. This will include SWF

(Simple Web Format) files to provide a virtual tour to leave visitors with a realistic way to travel through the site, helping to navigate through

the premises (on and offline).


Structue: The structure will display a template used to take different portals while browsing the JCC site. This should be easily navigatable

(users will have the option to go to areas directly) and readable (all text on the site will be clearly visible) for all visitors.


Media Types: Different media types will be utilized in the creation of the JCC site. These will include (but not limited to): text/html, text/css,

text/plain, image/gif, image/jpeg, image/png, image/svg+xml, Simple Web Format (SWF), audio/mp4, video/mpeg, video/mp4, application/json,

and application/javascript files.
